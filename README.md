# MPC² Simone I-U Plotter (Projekt 8)

Streamlit-Cloud-Version. Plottet Stromdichte vs. Potential aus ASC-Files,
mit Glättung, Datenreduktion, frei wählbaren Achsen, OCP/RPP-Markern, DE/EN-Switch.

**Lokal-Alternative ohne Installation:** siehe `standalone/Simone_IU_Plotter.html` —
einfach im Browser öffnen, kein Python nötig, läuft offline.

## Run locally
```
pip install -r requirements.txt
streamlit run app.py
```

## Deploy
Streamlit Cloud · Python 3.11 (runtime.txt) · Secret `password`.
